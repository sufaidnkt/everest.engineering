Offer Criteria
    The offers can be used according to the criteria that Tombo has captured in this table. Discount will not be provided if no offer code is applied.


            Distance (in km)    Weight (in kg)    
OFR001

OFR003

Challenge
    You are required to build a command line application to estimate the total delivery cost of each package with an offer code (if applicable).

Note : Note: Code should be extensible & scalable for more offer codes.


Delivery Cost = Base Delivery Cost + (Package Total Weight * 10) + (Distance to Destination * 5) 

Input Format

Input Format                                            Sample (I)
-------------------------------                 --------------------------------------           
base_delivery_cost no_of_packges                            100 3
pkg_id1 pkg_weight1_in_kg distance1_in_km offer_code1       PKG1 5 5 OFR001
pkg_id2 pkg_weight2_in_kg distance2_in_km offer_code2       PKG2 15 5 OFR002
pkg_id3 pkg_weight3_in_kg distance3_in_km offer_code3       PKG3 10 100 OFR003


Output Format                                            Sample (I)
-------------------------------                 --------------------------------------           
pkg_id1 discount1 total_cost1                       PKG1 0 175
pkg_id2 discount2 total_cost2                       PKG2 0 275
pkg_id3 discount3 total_cost3                       PKG3 35 665


Explanation 

    PKG 1: Testcase
        Base.delivery cost: 100
        Weight: 5kg | Distance: 5km 
        Offer code: OFR001


    Delivery Cost = 100 + (5 * 10) + (5 * 5) = 175.00
    Discount = 0 (Offer not applicable as criteria not met)
    Total Cost = 175 - 0 = 175.00


    PKG 3: Testcase
        Base.delivery cost: 100
        Weight: 10kg | Distance: 100km 
        Offer code: OFR003

    Delivery Cost = 100 + (10* 10) + (100 * 5) = 700.00
    Discount = 5% of 700 = 35.00 (Offer applicable as criteria met)
    Total Cost = 700 - 35 = 665.00





Problem -2 : Delivery Time Estimation
===========================
Now all these packages should be delivered to their destinations in the fleet of vehicles Kiki owns. She has N no. of vehicles available for delivering the packages. 

Things to keep in mind:
    -   Each Vehicle has a on limit (L) maximum weight (kg) that it can carry.
    -  All Vehicles travel at the same and in the same route. It is assumed that all the destinations can be covered in a single route.

Delivery Criteria
    - The delivery can be made using the criteria illustrated below.
    - Shipment should contain max packages  vehicle can carry in a trip
    - We should prefer heavier packages when  there are multiple shipments with the same  no. of packages. 
    - If the weights are also the same, preference  should be given to the shipment which can  be delivered first.

Challenge
    Tombo, the geeky business partner obsessed with delivery metrics wants to maximise efficiency.
